# 🩺 DocTime Server – Doctor Appointment Backend API

A secure and scalable backend API for the DocTime Doctor Appointment Booking Platform.  
Built with **Node.js, Express.js, MongoDB, and JWT authentication (JOSE)**.

---

## 🌐 Live Server

- 🚀 Live API: https://doc-appoint-server-virid.vercel.app
- 💻 GitHub Repo: https://github.com/muhiuddinshanto/DocAppoint_server

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB (Native Driver)
- JWT Authentication (JOSE)
- CORS
- Dotenv

---

## 📌 Features

- 👨‍⚕️ Doctors API (Get all doctors with search functionality)
- 🔍 Doctor search by name and specialty
- 🔐 Secure doctor details route using JWT authentication
- 📅 Appointment CRUD system (Create, Read, Update, Delete)
- 👤 User-based appointment filtering
- 🛡️ JWT middleware protection for sensitive routes
- 🌐 CORS enabled for frontend integration
- ⚡ Fast and lightweight REST API
